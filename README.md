# PixelMovies 🎬

PixelMovies is a premium, high-fidelity movie discovery platform built with Next.js, React, and Tailwind CSS. It features a fully responsive cinematic UI and a custom animated "PM" splash screen synced with studio-quality audio.

## Features ✨

- **Cinematic Loading Screen**: Custom Netflix-style "PM" brush stroke animation and audio perfectly synced on load.
- **Fully Responsive**: Adapts seamlessly from massive desktop monitors down to standard mobile screens using a mathematically precise wrapper scaling system.
- **Premium Aesthetics**: Dark mode, glassmorphism, dynamic animations, and luxury styling.
